제이쿼리로 미니게임 만들기

속성을 가지고 있는지를 true 또는 false로 반환해주는 is()메서드,
조건을 만족할경우와 아닌 경우 실행문을 제어 할 수 있는 조건문 if()를 활용
1) 보물로 정한 element에 클래스를 주고 is()로 주어진 클래스가 있는지 구함
2) if()로 있을 때와 없을 때를 제어.

자바스크립트로 디지털 시계만들기

1) 날짜 객체 new Date()로 년/월/일/요일/시/분/초 반환
2) 숫자로 반환되는 요일을 문자로 변환, 12시가 넘으면 오후로, 0시는 12시로 변경
3) 시,분,초에서 한자리일 경우 '0'을 추가하여 두자리로 변경후 1~3 정리한 값 변수에 담기
4) 선택자.inerHTML을 활용해 값을 html 안에 대입
5) 1~4 값을 함수에 담고 setInterval()을 활용해 1초 마다 함수 실행

svg로 사진스케치 애니메이션 만들기

1) 일러스트로 라인그려서 svg코드 생성 후 html로 이동
2) stroke-dasharray로 전체 길이와 반복길이 동일하게 한번만 설정
3) stroke-dashoffset으로 시작점을 전체길이가 끝나는 지점. 숨겨져 안보이는 지점으로 설정
4) animation을 활용해 stroke-dashoffset으로 시작점을 0으로, 숨겨진 사진 보이게 처리

setInterval와 animation, left값 변경으로 자동으로 움직이는 슬라이드

접근성 - aria-hidden : 보이는 슬라이드만 읽히도록, aria-live : 재생 polite (하던 작업하고 변경 사항 읽어줌), 정지 off(읽지 않음)
1) 전체 부모에 role="region" 으로 범위 역할 지정. 어떤 범위인지 aria-roledescription="carousel"로 슬라이드 통틀어 범위 지정
2) 슬라이더 개별로 role="grop" 그룹 역할 지정. 어떤 그룹인지 aria-roledescription="slide"로 슬라이더 범위라고 지정
3) 페이지네이션, 이전, 다음 버튼들에 aria-controls 지정
페이지네이션 : role="grop"를 가진 슬라이드 개별 아이템 / 이전,다음 : role="grop"를 가진 슬라이드의 부모. 4)모든요소에 aria-label로 제목 지정

스크롤 이벤트를 활용해서 parallax Scrolling

1)초기화면 opacity 0, translate로 위치 제어, 스크롤 했을 때 가지는 클래스로 미리 위치 제어
2)addClass로 정해둔 클래스 추가해서 위치 제어
3)각 요소의 offset().top 값을 구하고
4)if()조건문을 활용해 scrollTop()값이 각 요소의 offset().top 값보다 크면 정해둔 클래스 추가
5)빨리 스크롤 하는 사용자를 위해, scrollTop()값에 일정 수치 더해서 빨리지나가서 못보는 일 없도록 제어
